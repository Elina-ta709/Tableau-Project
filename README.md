# data-visualization-project-tableau
This project is a data visualization dashboard created using Tableau.

The dataset contains information about employed persons, and the goal of this project is to analyze and present insights in a clear and interactive way.

In this project, I:
- Connected Excel data to Tableau
- Cleaned and organized the dataset
- Created visualizations to understand employment trends
- Built an interactive dashboard for better insights

This project is part of my learning journey to become a Data Analyst.
